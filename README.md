# Minesweeper 扫雷 

一个使用 **Python + NumPy + Pygame** 实现的完整扫雷游戏。支持自定义棋盘大小与雷数，涵盖经典扫雷的所有核心机制。

---

## 项目结构

```
Minesweeper/
├── board.py      # 棋盘生成层 — NumPy 随机布雷与数字计算
├── game.py       # 游戏逻辑层 — 无 UI 依赖的纯逻辑控制器
├── main.py       # 图形界面层 — Pygame 渲染与交互入口
├── README.md     # 项目说明
├── .gitignore
└── .vscode/
    └── settings.json
```

三层架构：

| 文件 | 职责 | 依赖 |
|------|------|------|
| `board.py` | 数据模型 — 雷区生成、数字计算 | `numpy` |
| `game.py` | 游戏逻辑 — 点击、展开、标记、胜负判定 | `board` |
| `main.py` | 图形界面 — 棋盘渲染、鼠标交互 | `board`, `game`, `pygame` |

---

## 环境与运行

### 依赖

- Python 3.x
- NumPy
- Pygame

```bash
pip install numpy pygame
```

### 启动

```bash
python main.py
```

启动后在终端输入行数、列数、雷数，即可打开图形窗口开始游戏。

也可以单独运行各模块测试：

```bash
python board.py   # 终端打印随机棋盘
python game.py    # 终端演示点击/标记操作
```

## 游戏玩法

| 操作 | 效果 |
|------|------|
| **左键点击** 隐藏格 | 翻开格子 |
| **左键点击** 已翻开的数字格 | 若周围旗子数 = 该数字，自动翻开其余邻居（和弦操作） |
| **右键点击** 隐藏格 | 插旗 / 取消插旗 🚩 |
| 所有非雷格翻开 | 胜利 🎉 |
| 翻到雷 | 游戏结束 💥 |
